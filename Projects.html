Mini-sudoku project

Mini-sudoku project is a smaller version slightly smaller version of the full-size 9x9 sudoku.

Mini sudoku has a 4x4 grid with 4x4 squares, yet the complexity of the game stays relatively same.

The purpose of the project was to create a functional mini-sudoku game. With a customizable player name, score system, scoreboard

My part in the project was to make the functionality of the score table for other features as well.

I was the equipment manager in the project. I acquired the necessary components for the project, such as the LCD screen and the buttons.

Banksimul project

The banksimul or ATM-simulator was a program done using mostly object oriented programming

The program it self included a RFID reader and a few of keys to go along with the reader. The keys simulated the fysical bank card.

The program could read the individual cards and they were included in their own bank account. There was a database that held all of the cards and customer information

My part in the project was to make the functionality of the ATM, such as withdrawing money, checking the balance, and reading the bank card.

Along with the features for the atm's functionality, I did make most of the backend of the project myself

The project was challenging, but very fun to do.

Film Club project

The Film Club project or (Elokuvakerho) was a project where we created a website using react, where you can search for movies and see their information.

The project used an external database where the information of the movies / series was read from. In the project we also created a simple database of our own,

where we stored the information of the users who had registered, which user liked which movie and more on the site.

My part of the project was to create and modify the backend for our custom stuff we wanted to add to our website. I also was part of creating and modifying the frontend when new stuff / content was added

Country project

The Country was a project where we created a application using Android Developer's tools to create a simple app that could read data read data of off the internet and parse it into the ways we wanted.

We used the restcountries api to get the flag, population, and other information for the countries. We then parsed the data into nicely organized parts, where you could search the country and get it's information

My part of the project was to create the search functionality and the parsing of the data.
